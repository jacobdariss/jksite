FAQ

Questions fréquentes

Quelle différence entre Baobab et Fondation ?+
Baobab = hébergement classique avec domaine .SN. Fondation = PaaS managé multi-stacks pour déployer vos applications.
Qu'est-ce que le SLA contractuel ?+
Engagement écrit de 99.9% de disponibilité. Si le seuil n'est pas atteint, des pénalités s'appliquent.
Puis-je scaler à la demande ?+
Oui, avec Savane. Cloud privé isolé, VPC dédié, scalable sans interruption.
Qu'est-ce qu'un référent dédié ?+
Un interlocuteur technique nommé qui connaît votre contexte, avec appels mensuels proactifs et point d'escalade direct.
Je viens d'un autre hébergeur, comment migrer ?+
Onboarding Accompagné : audit, migration zéro downtime, configuration, formation. 50 000 FCFA forfait unique.
Fondation supporte quels langages ?+
Node.js, Python, PHP, Java. Déploiement Git intégré, backups quotidiens, SSL automatique.
